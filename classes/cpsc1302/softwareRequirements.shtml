Software Requirements

To complete all lessons, assignments, labs, and tests, you will need to access a computer with the following installed:

  1. Operating System (need at least one):
    1. Linux (try Lubuntu)
    2. Windows 7/8.x
    3. Mac OSX
    4. NOTE: Regardless of the OS you choose, you'll almost certainly need to set the appropriate environment variable(s). For Linux/Unix and MacOS systems this will probably be in your own shell run command file (e.g., .bashrc), while on Windows it's a bit different. This tutorial should help.
  2. Web Browser (need at least one):
    1. Chrome
    2. Firefox
    3. Internet Explorer
  3. Environment (recommended if using Windows):
    1. Cygwin provides a Unix-like environment on Windows, including many programming languages (e.g., C, C++, Lisp, Prolog). It's highly recommended that you install this as it will include some of the items from the next two sections below). NOTE: The full install will take a long time (e.g., 24 hours).
  4. Java Platform, Standard Edition (JSE)
  5. Java IDE (need at least one):
    1. jGRASP
    2. Eclipse
    3. And for those who just can't stand using an IDE, the following may be helpful:
      1. Vim for text editing
      2. Ant for application builds
  6. Code Source Versioning (recommended):
    1. Subversion server
    2. TortoiseSVN web client
  7. Java Source Code and Program Files from the textbook materials
