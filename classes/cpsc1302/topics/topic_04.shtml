  1. Two different kinds of conversions are illustrated here:
    1. Convert a String to a number.
    2. Convert one numeric value to another numeric value.
  2. Java program to convert a temperature on the Fahrenheit scale to one on the Celcius scale. Put the following code in a file named "ConvertTemp.java". 
    import java.io.*;

class ConvertTemp {
	public static void main(String args[]) {
		float farhenheit = Float.parseFloat(args[0]);
		System.out.println(farhenheit + "F"
					+ " = "
					+ (farhenheit - 32) * 5 / 9 + "C");
	}
}
  3. To pass command line arguments be sure to run the program using "java ConvertTemp -40", where "-40" is the temperature to be converted.
  4. Things to try:
    1. Change the program to convert Farhenheit to Kelvin.
    2. Write a new program to change US Dollars into Euro. Google "currency exchange rates dollar vs euro" to find today's rate of exchange.
