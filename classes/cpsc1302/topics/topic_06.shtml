Dr. J Dana Eckart: Computer Science 2 (CPSC 1302) - Comments & Javadoc

  1. Java supports two different comment types:
    1. Comments that start with "//" (two forward slashes) end at the end of the current line of text.
    2. Comments that start with "/*" (forward slash + asterisk) can extend over multiple lines, but only end when "*/" is seen.
  2. While the first type of comment can be nested, the second cannot be. Why not?
  3. A special form of the second type of comment (/* ... */) is called javadoc and can be used to create web pages for program documentation.
  4. Use the following at the top of each file: 
    
/**
 * Responsible for ... Used by ...
 *
 * @author Dr. J Dana Eckart
 * @version 2014-02-23
 */
public class MyClass { ...
