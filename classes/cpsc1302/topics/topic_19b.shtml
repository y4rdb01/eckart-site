Dr. J Dana Eckart: Computer Science 2 (CPSC 1302) - Vehicle Example

  1. While perhaps not as obvious as the biology taxonomy, vehicles also have a taxonomy that we tend to use (even if we don't write it down in a formal way).
  2. Define a set of Java classes for the following, extending the appropriate ones to achieve the hierarchy relationships commonly used. Note that the following are mixed up a bit, so you should probably read through the list first before starting to write any class definitions.
    1. Submarine
    2. Truck
    3. Spacecraft
    4. Airplane
    5. Quadracycle
    6. Helicopter
    7. Boeing747
    8. Aircraft
    9. Watercraft
    10. SurfaceVessel
    11. Car
    12. Destroyer
    13. Vehicle
    14. SpaceShuttle
    15. Landcraft
    16. Soyuz
    17. Bicycle
    18. Blimp
  3. Create an "identify" method as we did with the Animal example and override it as appropriate in any other classes that need it.
  4. Write "main" method inside a Driver class that creates an instance of each class and prints out its identification.
  5. Create a "primaryCharacteristic" method that returns a String which is describes the most distinguishing characteristic of that type of vehicle (e.g., "travels underwater"). If you cannot think of one, then return an empty String.
  6. Adjust the "main" method to print out both the identification and the corresponding distinguishing characteristic.
  7. Add 2 new types of vehicles, your choice, and update your "main" method to create instances of these new vehicles and print out their identification and corresponding distinguishing characteristics.
