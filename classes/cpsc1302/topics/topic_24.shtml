Dr. J Dana Eckart: Computer Science 2 (CPSC 1302) - Output Formatting

  1. java.io.PrintStream has printf (identical to format) for fine control of output.
  2. printf (and format) are far more flexible than print and println and no harder to use.
  3. Simplest use is outputStream.printf(formatString, arg_1, ... arg_n); where the formatString contains the format specifications (along with other text) and the args are the corresponding values (in order) to be printed according to those specifications.
  4. A format element:
    1. Begins with a "%" and ends with a format type (e.g., "d" for integer, "f" for floating point, "s" for string, "n" for newline).
    2. Between these 2 elements are a flag and a width. Some common flags are:
      -Print left-justified
      +Show plus sign for positive numbers
      0Print leading zeros
      ,Show 1000s separators
      (Enclose negative values in ()
