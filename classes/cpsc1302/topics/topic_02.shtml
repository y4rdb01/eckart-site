Dr. J Dana Eckart: Computer Science 2 (CPSC 1302) - Edit, Compile, Run, Debug

  1. Use a text editor to create a file named "HelloWorld.java" that contains the following Java source code: 
    import java.io.*;

class HelloWorld {
	public static void main(String args[]) {
		System.out.println("Hello World.");
	}
}
    Note that the name of the file (before ".java") should match the class name given in the source code.
  2. Compile the code using "javac HelloWorld.java".
  3. Run the compiled program using "java HelloWorld".
  4. If there were problems with the output, then you would
    1. Generate several hypothesies about what the problem might be.
    2. Run the program with different inputs to test each hypothesis, discarding those that turn out to be invalid, and perhaps generating new hypothesies in the process.
    3. When one or more hypothesies are supported by the available data, then devise adjustments to the program design and code to fix the problem(s) and return to step 1 (using the editor to adjust the program source code).
  5. Things to try:
    1. Alter the program to print out a different message.
    2. Alter the program to print out multiple messages, each on a different line.
    3. Without referring back to this code, can you create a program from scratch that prints out your name?
