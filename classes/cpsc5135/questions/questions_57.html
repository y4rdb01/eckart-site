• Write a NetLogo program that uses a 3-dimensional patch universe. Extend the SRDP model so that it works in 3 dimensions rather than just 2 dimensions.

  • Write a NetLogo program that uses a default 2-dimensional patch universe. Each patch should randomly be assigned the value of 0, 1, or 2 (each value being equally likely). A single turtle starts in the middle of the universe, heading north. During each time tick:
    • Based on the patch value the turtle is in, it turns:
      • left 90 degrees if the patch value is 0,
      • 0 degrees and continues in its current direction if the patch value is 1, and
      • right 90 degrees if the patch value is 2.
    • Then the turtle moves forward one unit.
