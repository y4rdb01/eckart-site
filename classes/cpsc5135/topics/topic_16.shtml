Dr. J Dana Eckart: Programming Languages (CPSC 5135) - Type Inference

  1. Not everything is declared with a specific type, so type inference infers types from the information that is given. For example, an expression given for the condition in an if-then statement should be of type boolean but the expression could involve lots of arithmetic operations and comparisons (along with boolean operations like and, or, and not).
  2. Type checking often depends on type inference (since the results of expressions must be inferred).
  3. [ Students are not expected to know the details of the ML Type Inference algorithm, but reading through section 6.3.2 is encouraged. ]
