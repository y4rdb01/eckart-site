Dr. J Dana Eckart: Programming Languages (CPSC 5135) - Using Smalltalk-X

  1. To start Smalltalk on the machines in the CCT 450 Lab, boot them into Windows 7, then start "C:\stx\projects\smalltalk\stx". This will (eventually) bring up 2 windows: ST/X Launcher, and Getting Started (which only appears the first time you launch stx). This is an interactive mode that allows you to browse classes and test code fragments (within the workspace).
  2. To run code fragments:
    1. Open a workspace using the ST/X Launcher's "Tool" menu and selecting "Workspace".
    2. Type (or paste) a test code fragment into the workspace (e.g., "Transcript show: (5 squared); cr").
    3. Highlight the code fragment and then execute it by clicking the thunderbolt icon (or by using the "DoIt" choice in the "Execute" menu).
  3. To run your code from a file (file should use ".st" extension by convention), you'll need to:
    1. Start the file browser from the "ST/X Launcher" using the "File" menu and choosing the "File Browser".
    2. In the file browser, navigate to the folder where your source files are located and highlight the desired file.
    3. Under the file browser's "File" menu, choose the "FileIn to Namespace...". Use the "Smalltalk" namespace (it should already be selected by default) and click "OK". NOTE: Any errors in the file will be indicated in the ST/X Launcher (which is the Transcript window). Make sure your code loads cleanly before continuing!
    4. Open a workspace (see above) and test your class(es) with code fragments (e.g., "Example new doSomething: 'Hello'").
  4. Test your understanding of the above by answering these self-assessment questions.
