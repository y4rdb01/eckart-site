Dr. J Dana Eckart: Programming Languages (CPSC 5135) - Java History

  1. Envisioned as the programming language for (TV) set-top boxes (similar to Roku, Apple TV, Chromecast, and Amazon Fire TV ).
  2. Work started in 1990 as a reimplementation of C++ (which was then about 10 years old), but quickly changed into a new language design/system known as Oak.
  3. With the emergence of the web (and the graphical web browser Mosaic in 1993), Oak switched focus from set-top boxes to web client programming language (using applets). The language was official renamed Java in 1994.
  4. Java began supporting the development of transactional and server based systems in December 1999 with the release of J2EE 1.2. This was done primarily through the creation of additional libraries and systems built using the existing language and library facilities.
  5. Java consists of:
    1. a programming language (Java);
    2. the Java Virtual Machine (JVM);
    3. Java Bytecode that is generated by Java compilers and executed by the JVM; and
    4. and a wide collection of libraries (both from Sun/Oracle and from numerous independent sources).
