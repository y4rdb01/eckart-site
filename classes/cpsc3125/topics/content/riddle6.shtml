Riddle Answer

find   .   -type   f   -printf   "%s   %p\n"   |   sort   -n   -r   +0   -1   |   head   -1   |   cut   -f1

Locates the largest file in the current directory structure. Find uses the -printf option to make a list of file names and sizes. Sort then sorts the list, based on the file size, in descending order. Taking the first line of the sorted output and then the second item on that line gives the desired result.
