Riddle Answer

find   directory_name   \(   -type   f   -o   -type   d   \)   -printf   "%f\n"   |   sort   |   uniq   -d   |   wc   -l

Counts the number of duplicate file/directory names within a directory structure. Find is used to locate all files and directories and to print their names. You should note that the use of the -printf option is necessary since the -print option returns the relative pathname and not the basename. Then sort the different names so that duplicates can be removed before counting the remaining, now unique, names.
