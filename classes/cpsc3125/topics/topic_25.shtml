Dr. J Dana Eckart: Operating Systems (CPSC 3125) - Unix Command Exercises

  1. Capture your interaction with bc as you determine the the average of the numbers 2, 3, 5, 7, 11, 13, and 17 (for which you should get 8.28571428571428571428).
  2. Use the calendar to determine what day of the week 31 Decemeber 2050 will be.
  3. Start the editor and then suspend it.
  4. List all of the current processes (associated with a terminal).
  5. List the current (unfinished) jobs.
  6. Kill the editor job.
  7. Make a new directory named "Kumquat".
  8. Move into the new directory and create a file named "heck", that contains your full name on the first and only line in the file.
  9. Go back one directory level, and change the name of the new directory from "Kumquat" to "silly".
  10. Make a copy of directory "silly", called "sillier".
  11. List the contents of both "silly" and "sillier" using a single command.
  12. Determine what directory you are currently in.
  13. Remove both "silly" and "sillier" using a single command.
  14. Using pwd and df determine how much space is left on the disk upon which your account is located.
  15. Determine how much space your files take up.
  16. Use find to determine if there are any filenames beginning with "ps" in the directory /usr/man/man1.
  17. Use find and head to print the first line of each file under the directory /usr/man/man1.
  18. Use more to look at the file /usr/man/man1/yacc.1
  19. Save to the file "sortedUsers" a list of the lognames of those people currently logged in, sorted alphabetically.
  20. Examine the file "sortedUsers" and then remove it.
  21. Count the number of files in your home directory.
  22. Count the number of files in your home directory that contain the word "the".
