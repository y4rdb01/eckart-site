Dr. J Dana Eckart: Advanced Programming Languages (CPSC 6129) - Extended Example: Forth

  1. The book for this course, The Definitive ANTLR 4 Reference is very detailed and includes numerous examples of how to use the system. In an attempt to avoid repeating those examples, an extended example based on the Forth programming language will be used. However, please note that no attempt will be made to implement all of Forth, though enough will be done to enable useful computations and to give a substantial introduction to Forth (as well as to ANTLR 4).
  2. Why Forth? While it's an older language, dating back to the early 1970s, it's still in use today. Forth's most notable feature is that it's a stack based language, differentiating it from most (all?) of the languages you've likely come in direct contact with. It is certainly different from Lisp, Prolog, C++, Smalltalk, and NetLogo that are covered in CPSC 5135. Exposure to yet another programming language paradigm is a great opportunity.
  3. There are three published standards for Forth, from 1979, 1983, and 1994. While only a subset of Forth will likely be implemented as an extended example for this course, I'll endeavor to conform to the latest version as much as practical:
    1. Forth-79 Standard
    2. Forth-83 Standard
    3. Forth-94 Standard
    4. Standard Forth-94 Reference
    5. Forth-200x (RC2)
  4. Learning a language from a reference manual/standard is not generally very easy, so here are some on-line tutorials which should help you get acclimated quickly, making it easier to understand the example ANTLR 4 code for Forth:
    1. A Beginner's Guide to Forth
    2. Starting FORTH
    3. Forth Tutorial
    4. And so Forth...
    5. Programming Forth
    6. Len's Forth Tutorial
