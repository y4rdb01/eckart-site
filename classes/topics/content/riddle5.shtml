Riddle Answer

find   .   -type   f   -printf   "%T@   %p\n"   |   sort   -n   +0   -1   |   head   -1

Locates the least recently modified file in the current directory structure. Find returns the modify times of files along with the file names, by using the -printf option. These are then sorted with respect to the time, latest times first, thus giving the answer as the first line of the sorted list.
